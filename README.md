## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/my-anime-favorites.git
    ```
2. Navigate to the project directory:
    ```bash
    cd my-anime-favorites
    ```
3. Install the dependencies:
    ```bash
    flutter pub get
    ```
4. Run the app:
    ```bash
    flutter run
    ```
